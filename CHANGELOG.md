# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-01-30
### Added
- Advanced search and filters (text, date range, distance, duration)
- Workout notes with inline editing
- Custom dark activity map theme
- Training Analytics
  - HR Zones: Time-in-zone analysis based on configurable Heart Rate zones.
  - Training Load: Tracks fitness (CTL), fatigue (ATL), and form (TSB).
  - Recovery Score: Estimates readiness to train based on Training Stress Balance.

### Changed
- Performance charts layout and styling
- Theme consistency across activity detail sections

### Fixed
- Symmetry issues when showing three performance metrics
- Dark theme styling for map attribution and controls

## [0.1.0] - 2025-01-12
### Added
- Initial release
